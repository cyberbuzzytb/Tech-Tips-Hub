Windows Tips & Tricks

1. Quick Screenshot

Press Windows + Shift + S to capture a part of your screen instantly.

2. Dark Mode Shortcut

Press Windows + Ctrl + C to toggle color filters (if enabled, it can switch between light and dark modes).

3. Virtual Desktops

Use Windows + Ctrl + D to create a new virtual desktop and Windows + Ctrl + Left/Right to switch between them.
