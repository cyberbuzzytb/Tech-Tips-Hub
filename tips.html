Windows Tips

Windows 10 Tips

  • Windows Key + S: Opens the Search bar.
  • Windows Key + E: Opens File Explorer.
  • Windows Key + Tab: Opens Task View.
  • Windows Key + D: Shows the desktop.
  • Windows Key + M: Minimizes all windows.
  • Ctrl + Shift + Esc: Opens Task Manager.
  • Alt + Tab: Switch between open apps.
  • Windows Key + L: Lock your computer.
  • Windows Key + R: Opens Run dialog.
  • Windows Key + P: Opens Project screen.
  • Windows Key + I: Opens Settings.
  • Windows Key + X: Opens Quick Link menu.
  • Windows Key + Number (1-9): Opens pinned taskbar apps.
  • Windows Key + G: Opens Xbox Game Bar.
  • Windows Key + Shift + Left/Right: Moves window between monitors.
  • Ctrl + Shift + N: Creates a new folder.
  • Alt + F4: Closes the active window.
  • Windows Key + Ctrl + D: Creates a new virtual desktop.
  • Windows Key + Ctrl + Left/Right: Switches between virtual desktops.

Windows 11 Tips

  • Windows Key + C: Opens Microsoft Teams chat.
  • Windows Key + Z: Opens Snap Layouts.
  • Windows Key + A: Opens Quick Settings.
  • Windows Key + N: Opens Notification Center.
  • Windows Key + H: Opens voice typing.
  • Windows Key + K: Opens Cast menu.
  • Windows Key + Alt + R: Starts screen recording.
  • Windows Key + Shift + S: Opens Snipping Tool.
  • Windows Key + Ctrl + Shift + B: Restarts graphics driver.
  • Windows Key + Spacebar: Switch keyboard layouts.
  • Windows Key + W: Opens Widgets panel.
  • Windows Key + V: Opens Clipboard history.
